# 数据同步接口集成文档

## 概述

本文档记录了从数据管理平台（xai-vehicle-operation-manager）同步数据到BeeEval平台（xai-vehicledata）的接口集成工作。

同步数据包括：
- 功能域平均分（type=1）
- 三级指标分数（type=2）
- 基础能力指标分数（type=3）
- 开源题目分数

            .eq(VehicleDomainScoreEntity::getType, (short) 3));

**接口路径**: `POST /vehicle/domaintree/baseDomainIndexScoreSort`

**功能**: 基础能力指标排行榜，支持指定车辆或全量查询

**修改内容**:
- 删除硬编码的基础能力数据
- 集成同步的基础能力指标数据（vehicle_domain_score.type=3）
- 支持全量查询（不传brandBaseInfoId时查询所有车辆）
- 对旧数据进行单位转换（首字响应时长和图像生成速度从毫秒转换为秒，保留两位小数）
- 行业均值数据也进行单位转换

**核心逻辑**:
```java
// 全量查询逻辑

boolean queryAllVehicles = (vehicleIdsToQuery == null || vehicleIdsToQuery.isEmpty());

if (queryAllVehicles) {
    // 查询所有有同步数据的车辆ID
    vehicleIdsToQuery = vehicleDomainScoreDao.selectList(
        Wrappers.<VehicleDomainScoreEntity>lambdaQuery()
            .eq(VehicleDomainScoreEntity::getType, (short) 3)
            .select(VehicleDomainScoreEntity::getVehicleId)
            .groupBy(VehicleDomainScoreEntity::getVehicleId))
        .stream()
        .map(VehicleDomainScoreEntity::getVehicleId)
        .distinct()
        .collect(Collectors.toList());

        double value = ((Number) responseTime).doubleValue();
        double convertedValue = value >= 1000 ? value / 1000 : value;
        double roundedValue = Math.round(convertedValue * 100.0) / 100.0;
        row.put("451751102942019599", roundedValue);
    }

```

## 数据同步规则

### 同步数据类型
1. **type=1**: 功能域平均分 → vehicle_domain_score表
2. **type=2**: 三级指标分数 → vehicle_domain_score表
3. **type=3**: 基础能力指标分数 → vehicle_domain_score表
4. **开源题目分数**: → beeeval_open_case_score表

### 数据一致性保证
- 使用@Transactional确保数据同步的原子性
- 同步失败时支持事务回滚
- 通过vehicleId和domainId等字段保证数据唯一性

### 性能优化
- queryIndexScore接口从N+1查询优化为批量查询
- 批量获取车辆信息（getVehicleInfoBatch方法）
- 内存中进行数据过滤和组装

### 数据优先级
- 同步数据不覆盖原有业务数据
- 新车数据持续更新，旧数据继续展示
- 通过alreadyExists检查避免数据重复

### 单位转换
- 首字响应时长（domainIndexId=451751102942019599）：毫秒 → 秒
- 图像生成速度（domainIndexId=451751102942019601）：毫秒 → 秒
- 转换后保留两位小数
- 行业均值数据也进行相同转换
