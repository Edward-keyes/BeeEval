# 智能问答系统使用指南

## 📋 系统概述

基于通义千问大模型的智能问答系统，支持自然语言查询汽车智能驾驶评估数据，提供结构化答案和可视化展示。

## 🚀 快速开始

### 1. 配置环境变量

```bash
# 设置通义千问API密钥
export QA_LLM_API_KEY=your_api_key_here

# 设置数据库连接（如果需要）
export DB_USERNAME=your_db_user
export DB_PASSWORD=your_db_password
```

### 2. 启动应用

```bash
# 使用QA配置文件启动
java -jar xai-vehicledata.jar --spring.profiles.active=qa
```

### 3. 验证系统状态

```bash
# 检查健康状态
curl http://localhost:8080/actuator/health

# 检查LLM服务状态
curl http://localhost:8080/api/v1/qa/health
```

## 🔌 API 接口使用

### 创建会话

```bash
curl -X POST http://localhost:8080/api/v1/qa/session \
  -H "Content-Type: application/json" \
  -d '{
    "userId": "user123",
    "userName": "张三"
  }'
```

**响应示例：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "sessionId": "sess-abc123456789",
    "status": "active",
    "createTime": "2024-01-27T10:00:00",
    "lastActivityTime": "2024-01-27T10:00:00"
  }
}
```

### 提交问题查询

```bash
curl -X POST http://localhost:8080/api/v1/qa/query \
  -H "Content-Type: application/json" \
  -d '{
    "sessionId": "sess-abc123456789",
    "question": "查询所有车辆的功能域平均分",
    "userId": "user123",
    "userName": "张三"

```

**响应示例：**

## ⚙️ 高级配置

### 自定义问题模板

```bash

curl -X POST http://localhost:8080/api/v1/qa/templates \
  -H "Content-Type: application/json" \
  -d '{
    "name": "自定义查询模板",
    "description": "针对特定业务场景的查询",

  }'
```


## 🛠️ 故障排除

### 常见问题

#### 1. API调用失败
```
错误: 通义千问API调用异常
解决: 检查API密钥配置和网络连接
```

#### 2. 数据库连接失败
```
错误: SQL查询失败
解决: 检查数据库配置和连接状态
```

#### 3. 缓存服务异常
```
错误: 缓存操作失败
解决: 检查Redis连接配置
```

#### 4. 模板匹配失败
```
错误: 未找到匹配模板
解决: 检查模板配置或使用通用查询
```

### 日志调试

```bash
# 查看应用日志
tail -f logs/xai-vehicledata.log

# 查看特定级别日志
grep "ERROR" logs/xai-vehicledata.log
```
